# Changelog

All notable changes to RobotEvo will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

- Initial PyPI package structure
- `pyproject.toml` for modern Python packaging
- Comprehensive documentation

### Changed
- Improved README with installation and usage instructions

### Fixed

## [0.1.0] - 2024-01-12

### Added
- Initial release to PyPI
- Core EvoScriPy API
- Example protocols (demos, EVO 75/100/200)
- GUI interface for protocol selection
- Support for Freedom EVO 75, 100, and 200 robots
- Documentation at robotevo.readthedocs.io

### Known Issues
